## How to try for yourself
1. Clone the repository:
   git clone https://github.com/pilar4/PDM-paint
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python main.py
4. To draw lines use right click and mark two points (begining and end of a line)
